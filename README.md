# El-Kubo TCG

Landing page da El-Kubo TCG — loja especializada em Trading Card Games.

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Variáveis de ambiente

Crie um arquivo `.env` com:

```
NEXT_PUBLIC_WHATSAPP_PHONE=5541999999999
NEXT_PUBLIC_GROUP_URL=https://chat.whatsapp.com/SEU_LINK
```

## Build para produção

```bash
npm run build
npm run start
```
